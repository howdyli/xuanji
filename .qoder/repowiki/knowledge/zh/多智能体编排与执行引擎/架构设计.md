- **分层架构**：模块划分为配置层（`config/*.yaml` 定义角色与任务模板）、模型层（`models.py` 定义结构化输出）和执行层（`main_crew.py` 与 `skill_crew.py`）。
- **主从协作模式**：`MemoryAwareCrew` 作为主控节点，通过 `SkillLoaderTool` 动态加载能力；`build_skill_crew` 负责在隔离的沙箱环境中实例化子智能体执行具体任务。
- **生命周期管理**：采用工厂函数 `build_agent_fn` 封装智能体初始化逻辑，集成会话上下文恢复、历史消息压缩及异步索引功能。
- **钩子集成**：利用 `@before_llm_call` 和 `@before_tool_call` 装饰器实现调用前的参数标准化（如 Python 到 JSON 的类型转换）及会话状态同步。
